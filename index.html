"git add ."

Monitora os arquimos excluídos, adicionados ou modificados para serem versionados.


"git commit -m"

Cria a versão dos arquivos, a mensagem entre aspas deve representar o que foi feito.


"git push origin main"

Envia a versão criada para o github.


"git pull origin main"


"git clone -link do repositório-"

Clonar repositório do github


"git clone -link do repositório- +nome"

Clonar repositório do github para a pasta.


"git status"

Ver alterações no repositório.
